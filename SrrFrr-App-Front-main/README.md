# inD — Application Mobile

Application mobile iOS & Android de **inD**, plateforme VTC marocaine.

> Documentation complète (architecture, backend, déploiement) :  
> **[README principal](../README.md)**

## Démarrage rapide

```bash
flutter pub get
flutter gen-l10n
# Créer .env.local (voir README principal)
flutter run
```

## Prérequis

- Flutter SDK ≥ 3.9
- Fichier `.env.local` avec `API_BASE_URL` et `GOOGLE_MAPS_API_KEY`
- Backend API en cours d'exécution ([instructions](../README.md#backend))
